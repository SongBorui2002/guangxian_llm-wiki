# Transkoder 键盘快捷键

Transkoder 2025 常用键盘快捷键汇总。完整列表见用户指南附录 D。

## 导航

| 快捷键 | 功能 |
|--------|------|
| **Enter** | 进入 Node Page（节点页面） |
| **P** | 进入 Render Page（渲染页面） |
| **F12** | 显示完整快捷键列表 |

## Render Page

| 快捷键 | 功能 |
|--------|------|
| **Ctrl+R** | 启动标准渲染 |
| **P** | 暂停渲染 |
| **Esc** | 中止渲染 |

## Finalize & DCP

| 快捷键 | 功能 |
|--------|------|
| **Shift+D** | 打开 DCP/IMF Finalize 窗口 |

## Node Page / 模板

| 快捷键 | 功能 |
|--------|------|
| **Shift+Y** | 仅保存 Viewing Template |
| **Shift+T** | 保存 Grade 和 Viewing Template（覆盖其他片段） |
| **Ctrl+S** | 常规保存 |

## 时间线编辑

| 快捷键 | 功能 |
|--------|------|
| **I** | 设置 Reel Marker In 点 |
| **O** | 设置 Reel Marker Out 点 |
| **Ctrl+Backspace** | 删除选中 Reel |

## 通用

| 快捷键 | 功能 |
|--------|------|
| **F1** | 帮助 |
| **F12** | 快捷键列表 |
| **Ctrl+Z** | 撤销 |

> [!note]
> 完整快捷键列表请参考用户指南附录 D（`TKDUserGuide_dark__sections/32-appendix-d-keyboard-shortcuts.md`）。
