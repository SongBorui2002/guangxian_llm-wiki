# Transkoder 故障排查

常见问题诊断和修复指南。详细内容见用户指南附录 K。

## 请求支持时需收集的信息

在联系 COLORFRONT 支持之前，请准备以下材料：

1. **Transkoder 日志文件** — 问题发生后立即保存的日志
2. **崩溃转储文件** — Crash dump 文件
3. **项目导出** — 通过 File 菜单将有问题的项目导出为 `.zip` 文件
4. **样本文件** — 任何相关但非敏感的元数据或样本文件

## 调试工具

- **帮助 → 打开用户指南**：在软件中直接查看手册
- **F12**：显示完整快捷键列表
- **Help → Support → Troubleshoot Issues**：内置诊断工具
