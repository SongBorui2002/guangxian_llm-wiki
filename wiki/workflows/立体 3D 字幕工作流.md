# 立体 3D 字幕工作流

利用 DaVinci Resolve 和 StereoscopicSubtitlePlugin 创建符合 SMPTE DCDM 标准的立体 3D 字幕的完整工作流。

## 工作流概览

```
2D 字幕添加 → 渲染字幕视频（含 Alpha）→ 重导入为立体 3D → 
立体调整（Color Page）→ EDL 分段 → DCDM XML 生成
```

## 详细步骤

### 1. 添加 2D 字幕

- 在 DaVinci Resolve 立体 3D 项目中添加常规 2D 字幕
- 确保字幕内容和时间轴正确

### 2. 渲染字幕到视频

- 仅渲染字幕到带 Alpha 通道的视频文件
- 输出格式建议：ProRes 4444（保留 Alpha）

### 3. 重导入为立体 3D

- 将字幕视频作为立体 3D 素材重新导入
- 设置正确的立体模式

### 4. 生成 EDL 分段

- 使用插件的 EDL Generator 功能
- 根据字幕内容自动切割字幕视频轨

### 5. 立体深度调整

- 在 **Color Page** 上进行立体调整
- 使用 **Anaglyph（红蓝）立体预览模式**确认深度效果
- 对每个字幕位置通过 **关键帧** 设置汇聚值（Convergence）

### 6. 生成 DCDM 立体字幕 XML

- 使用插件的 XML Generator 功能
- 从字幕轨提取文本内容
- 从视频轨的汇聚关键帧换算 Zposition 值
- 输出符合 SMPTE ST 428-7 的 DCDM 立体字幕 XML

## 验证

| 工具 | 用途 |
|------|------|
| **DCDMSubtitle-2014.xsd** | XML Schema 验证 |
| **DCP-o-Matic**（v2.18.15+） | DCP 封装和 VariableZ 验证 |
| **EasyDCP Player** | 播放验证（注意：忽略 VariableZ） |

## 插件安装

```bash
# 复制插件到 DaVinci Resolve 工作流集成目录
cp -r StereoscopicSubtitlePlugin/ \
  "/Library/Application Support/Blackmagic Design/Davinci Resolve/Workflow Integration Plugins/"
```

重启 DaVinci Resolve → `Workspace → Workflow Integrations` 调用。

## 已知限制

- VariableZ 在 SMPTE 规范中为可选参数，部分播放器忽略
- 插件处于早期开发阶段，结果需仔细验证
- 建议在影院硬件上进行实际放映测试
