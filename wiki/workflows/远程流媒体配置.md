# 远程流媒体配置

Transkoder 2025 内置参考质量流媒体播放服务器，实现安全的远程视频监控。工作流可通过 REST API 和自定义 Web 表单自动化。

## 系统架构

```
Transkoder 工作站 ──流媒体──→ 远程瘦客户端
      (本地/云端)              (浏览器访问)
```

## 远程流媒体特性

| 特性 | 说明 |
|------|------|
| **参考质量** | 视频流保持参考级图像质量 |
| **高安全性** | 流媒体传输加密 |
| **瘦客户端** | 客户端仅需浏览器，无需安装软件 |
| **REST API** | 渲染任务可通过 REST API 触发和自动化 |
| **Web 表单** | 自定义 Web 表单简化操作流程 |
