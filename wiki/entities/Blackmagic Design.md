# Blackmagic Design

**Blackmagic Design** 是全球领先的专业视频硬件和软件制造商，由 Grant Petty 担任 CEO。

## 核心产品线

| 产品 | 说明 |
|------|------|
| **DaVinci Resolve** | 旗舰级一站式后期制作软件（编辑 + 调色 + Fusion 特效 + Fairlight 音频） |
| **Blackmagic Cinema Camera** | 专业数字电影摄影机系列 |
| **ATEM** | 现场制作切换台系列 |
| **DeckLink** | 采集卡和回放卡 |
| **UltraStudio** | 外部采集和回放设备 |
| **DaVinci Control Panels** | 专业调色台硬件 |

## DaVinci Resolve 发展历程

DaVinci 最初是独立的高端调色系统，被好莱坞广泛用于电影调色。Blackmagic Design 收购后将其发展为集成式后期制作平台，融合了剪辑、合成、音频和交付功能，并推出了功能完整的免费版本。
